Суть метода наименьших квадратов

Метод наименьших квадратов позволяет найти прямую y = a·x + b, которая лучше всего аппроксимирует набор точек (xi, yi). Формулы для коэффициентов:

a = (N·Σ(xi·yi) - Σxi·Σyi) / (N·Σ(xi2) - (Σxi)2)

b = (Σyi - a·Σxi) / N

Где N — число точек, а Σ обозначает суммирование по всем точкам.

Алгоритм в программе:

  1. Считать массив DataX и DataY.
  2. Вычислить суммы: Σx, Σy, Σx², Σ(x·y).
  3. Подставить в формулы и получить a и b.
