Как пользоваться программой

1. Загрузка данных

  1. Нажмите Файл → Открыть или кнопку "Открыть" на панели.
  2. Выберите INI-файл с данными в формате: 
    [Data]
X=1,2,3,4
Y=2,4,5,4
  3. Или введите значения вручную в таблицу слева.

2. Вычисление

Нажмите кнопку Вычислить. Коэффициенты появятся в полях "Коэфф. a" и "Коэфф. b".

3. Сохранение результатов

Нажмите Файл → Сохранить или кнопку "Сохранить" и выберите путь. Текущие точки и коэффициенты сохранятся в INI.

4. Анимация

Чтобы наблюдать динамическое построение прямой, отметьте чекбокс "Анимация" до вычисления.
